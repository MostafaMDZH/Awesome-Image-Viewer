Image Viewer

V1.0.47

React, Angular, Vue, and Typescript compatible image viewer

# Features

  • Lazy Loading
  • Zoomable
  • Custom Buttons
  • Description
  • Custom Select
  • Swipe On Touchscreen
  • Cover Size
  • Thumbnail Support
  • Navigation With Arrow Keys
  • Responsive Design
  • Dynamic HTML
  • Lightweight
  • Zero Dependency
